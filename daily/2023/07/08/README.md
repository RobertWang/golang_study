# Go 每日一题

> 今日（2023-07-08）的题目如下

下面代码中，x 已声明，y 没有声明，判断每条语句的对错。

```golang
x, _ := f()
x, _ = f()
x, y := f()
x, y = f()
```

<details>
<summary style="cursor: pointer">🔑 答案解析：</summary>
<div>

参考答案及解析：错、对、对、错。

知识点：变量的声明。

1. 错，x 已经声明，不能使用 :=；
2. 对；
3. 对，当多值赋值时，:= 左边的变量无论声明与否都可以；
4. 错，y 没有声明。
